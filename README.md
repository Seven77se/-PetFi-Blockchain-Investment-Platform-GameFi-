## 📌 Project Introduction | 项目介绍

This project is a **Pet-themed blockchain investment platform**, combining GameFi mechanics with financial growth models, designed for high-engagement user acquisition and monetization.

该项目是一套 **宠物养成 + 区块链收益系统（PetFi 模型）**，通过游戏化互动 + 投资机制，提高用户留存与转化率，适用于海外市场运营。

✔ 宠物养成 + 收益结合
✔ 区块链资产体系（USDT）
✔ 多语言支持（全球市场）
✔ 可快速上线运营

---

## 🛠 Tech Stack | 技术架构

* **Backend（后端）**: Java (Spring Boot)
* **Frontend Web（管理端）**: Vue2 + Element UI
* **Mobile App（用户端）**: UniApp（支持 H5 / App）
* **Database（数据库）**: MySQL
* **Architecture（架构）**: 前后端分离

## 💡 Core Features | 核心功能

### 🐶 Pet System 宠物系统

* 宠物认养 / 购买
* 宠物等级成长体系
* 不同宠物对应不同收益率
* 宠物升级 / 生命周期管理

---

### 💰 Financial System 资金系统

* 数字货币充值（USDT）
* 提现（TRC20 / BEP20）
* 收益自动计算
* 余额与资产管理

---

### 🔗 Blockchain Wallet（自研钱包系统）

* 自研钱包 DApp
* 地址生成 / 管理
* 区块链充值监听（自动到账）
* 自动归集（Collect）
* 自动转账（Transfer）
* 支持 TRON / BSC 网络

---

### 📊 Earning Mechanism 收益机制

* 固定周期收益模型（如 7天 / 15天 / 30天）
* 自动收益发放
* 收益可视化展示

### 👥 Referral System 裂变系统
